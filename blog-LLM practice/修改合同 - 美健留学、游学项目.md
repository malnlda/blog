
#  LLM 使用经验：修改合同

今天使用大语言模型（例如 DeepSeek 等，通称 LLM）修改了一份合同，做个记录。

第一步：为模型设定身份，概述合作背景，发送脱敏后的合同初稿全文，提出修改需求，并请求模型对合同提出修改建议。不应在第一次对话就要求模型修改合同，而是请求模型结合项目背景、合同初稿和修改需求提出详细的修改建议，让模型「多想一些」。

提问（1）

```

回答（1）

```

以下建议旨在帮助乙方（以及甲方）进一步完善和修改现有合作协议文本，使之更具可执行性和公平性，并减少未来潜在争议。基于协议原文，我从律师角度列出以下修改和完善建议，供参考：

---

一、明确合作项目与合作模式 

1.项目范围 

建议在协议第一条中，进一步明确“留学项目”或“合作项目”的具体内容和范围。例如：

- 甲方与乙方共同协助海外高校来华进行交流、宣传活动；
- 甲方负责搭建并执行留学服务流程，包括学生咨询、申请、签证、入学手续等；
- 乙方负责在当地对接商业中心、社区，办理政府审批，进行活动落地等。
- 如有除文中提到的寒暑假研学团、海外高校留学等更多合作形式或模式，应在协议中列明或加以说明（例如夏令营、冬令营、访学项目等）。

2.项目排他性或非排他性

明确约定乙方是否可以同时与其他类似项目合作，甲方是否仅与乙方作为独家合作伙伴展开大湾区市场拓展。若无排他要求，应在协议中说明该合作为非排他型，避免后续争议。

二、双方的权利与义务 

（一）甲方责任和义务 

1.资源提供的具体内容*

协议中虽提到甲方提供“社区商场、院校联系方式、市场推广策划、活动设计及搭建、研学生源等”，但建议在协议附件或条款中列明具体的资源清单或提供方式及时间安排。例如：

- 甲方应在活动开始前多少天提供场地对接信息；
- 甲方应在活动期间安排专门的市场推广人员 与乙方对接；
- 甲方提供的“研学生源”是通过何种渠道获取（自有招生渠道、合作院校推荐等），并在何时提供给乙方；
- 如果某些资源无法按时或无法足额提供，甲方是否需要承担违约责任或补偿。

2. 专人对接与合作协调 

建议明确甲方指定的专门对接人信息（姓名、联系方式）以及其授权范围和工作职责，比如活动协调、资金结算、问题处理机制等。

3. 政府审批与风险承担 

鉴于项目在不同地区需要办理相应政府审批或备案，建议协议中进一步明确：

- 甲方有义务配合乙方提供必要的资质文件（如办学资质、海外高校合作授权文件、留学服务资质证明等），以完成政府审批手续；
- 若甲方未提供相应材料或无法配合导致审批失败或延期，应如何处理；
- 是否需要双方共同承担审批费用，或由哪一方承担。

（二）乙方责任和义务 

1. 活动落地与商务对接 

建议细化乙方“争取项目落地”的职责范围，例如：

- 在哪些城市或地区对接商业中心、居民社区、教育局、会展场地等；
- 需要在何时完成场地预订、活动申请等工作；
- 对接失败或政府审批未获通过时的责任分担及应对机制。

2. 市场推广与宣传 

若乙方也参与活动现场宣传、招募或其他市场推广事宜，建议在协议中补充乙方需要承担的推广方式、渠道，以及推广成本由谁负责或如何分摊。

3. 保密与数据使用 

建议增加保密条款，明确活动中或后续乙方所接触到的学生、家长、院校信息等均需要保密；任何一方不得在未取得对方同意的情况下向第三方披露商业机密或个人信息。

三、项目收益分配与费用核算 

1. 收入构成与定义 

协议指出甲方收入来源包括展位费、留学服务费、顾问费等；但目前仅约定乙方享有50%利润分配。为避免纠纷，建议明确：

- “净利润”的计算公式及范围，列明扣减项：“活动收入减去雇佣人员工资，支付社群拥有者的费用，市场推广费，设计及搭建费等” 。
- 在协议中列出典型支出项目清单，以及如何确认支出金额（如需提供对应发票或核算单据等）。
- 双方可对“支出项目”设置封顶或范围，防止一方随意扩大成本支出，减少另一方分成。

2. 费用分担与承担 

若乙方为协助落地工作也会产生某些实质性费用，如差旅费、政府审批费用等，建议明确约定这些费用是否包含在甲方列支的成本中，还是由乙方自行承担。

若项目活动经费由双方共同先行垫付，也应说明各自垫付比例、结算周期和方式。

3. 结算周期与支付方式 

建议详细约定收益结算的周期（例如每次活动结束后30天内结算、每季度结算或按约定频次结算）、结算流程（是否需要双方对账、确认收入及支出），以及甲方支付乙方利润分成的具体方式（银行转账、支付时间、支付违约责任等）。

4. 税费处理 

建议明确双方各自应承担的税费义务（增值税、个人所得税、企业所得税等），以及开具发票或税务处理的相关规定，避免后期税务风险。

四、合同期限与续约/变更/终止 

1. 合同有效期与续签安排 

协议第四条写明：从2025年3月__日起至2025年12月31日止 ，并在合约生效后每半年可作协商。建议将“每半年可作协商”具体化：若要对项目规模、城市范围、收益分配方式等进行调整，应通过书面形式（如补充协议）进行变更。

2. 终止与违约 

第五条仅指出若双方“发生重大的利益冲突或破坏协议之诚信，可以终止合作”，建议补充：

- 定义“重大利益冲突”或“违约行为”的具体情况（如一方不按约定提供资源、拖延支付或擅自泄露商业机密等）；
- 终止通知程序（如应提前多少天书面通知对方）；
- 违约责任或补偿机制（如已产生的活动成本如何分担、已收取的报名费或展位费如何处理，是否要退还或结算，等等）。

建议增加不可抗力条款，若因自然灾害、疫情、政策变化等导致活动无法正常开展或延期，应当如何处理。

五、知识产权与保密 

1. 品牌使用与宣传 

若甲方或海外院校的名称、logo、商标等需要在活动宣传物料中使用，建议在协议中明确授权方式、使用范围及规范，避免产生侵权或纠纷。

如乙方也有自有品牌或渠道，需要双方共同使用、宣传，也宜在协议中明确各自知识产权的归属和授权。

2. 保密义务 

留学行业中，客户名单、合作院校信息往往是核心资源，建议在协议中明确保密义务，包括：

- 对外合作宣传或活动介绍中，允许公开的信息范围；
- 乙方可否在后续与第三方合作中提及本项目的合作经验；
- 双方应当对学生或家长的个人隐私信息严格保密，并遵守相关隐私保护法律法规。

六、争议解决 

1. 争议解决方式 

协议中建议增设争议解决条款，如双方发生争议应优先通过协商解决；如协商不成，可选择向特定有管辖权的法院起诉，或约定由特定仲裁机构进行仲裁。

若项目主要在某个城市进行，宜在此城市或甲乙双方约定的地点进行诉讼或仲裁，避免后期为管辖权问题互相推诿。

2. 适用法律 

建议写明本协议适用的法律法规（如中国大陆法律），确保对合同效力以及争议处理有明确法律依据。

七、附则与其他 

1. 合同文本与附件 

建议对于项目具体的执行方案、活动计划、时间表、双方人员名单、经费预算等内容，采用附件或补充协议形式，并在主合同中注明附件是合同不可分割的组成部分。

可以在协议最后增设“与本协议相关的附件”为“附件一”“附件二”等，并明确附件与本协议具有同等法律效力。

2. 生效条件 

目前协议中没有明确写明何时开始具备法律约束力，或是否需要双方签字盖章。

建议在合同末尾添加：“本协议自双方签署并加盖公章之日起生效”，以防后期就“是否生效”产生争议。

3. 签约主体与签约代表 

需确保甲方、乙方均以合法主体名义签署协议，并由有权代表（法定代表人或授权代表）签字，盖公司公章或合同专用章。

总结 

综合而言，现有协议文本较为简略，许多核心事项（如费用结算方式、支出核算依据、违约责任、知识产权与保密、争议解决等）尚不够清晰。为确保本次合作顺利推进并保障双方合法权益，建议就以上各点进行详细修订。可通过补充协议 或在正文中逐条完善相应条款，以免在后续合作过程中产生不必要的纠纷。

温馨提示：以上建议基于一般商业合同实践及律师经验，不构成对具体事项的绝对保证。各方应结合自身实际业务情况，与专业律师及财务人员进一步核实、调整并最终落入文字，以形成合法、完整、可执行的正式合约文本。

3.2 乙方应保存所有相关票据及证明文件，以便与甲方对账。

4. 保密与信息安全 

4.1 乙方承诺对从甲方处获得的院校信息、学生及家长资料等保密，不得向无关第三方泄露；
4.2 任何有关学生个人隐私或商业机密的信息均应遵守国家相关法律法规并妥善保管。

第四条　收益分配与费用结算 

1. 项目收入构成 

1.1 本协议所称“项目收入”，包括但不限于：

  - 参展或展位费；
  - 家长支付的留学咨询与服务费用；
  - 海外合作院校（或机构）支付的顾问费、推荐佣金；
  - 研学团团费利润；
  - 其他经双方确认的收入（如：品牌推广收益、赞助费等）。

2.1 双方确认，“净利润”或“纯利”指：项目总收入-必要直接成本

2.2 上述各项支出须提供合法票据或经双方认可的费用明细并以书面形式确认。
